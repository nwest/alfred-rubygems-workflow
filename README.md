# Alfred Rubygems Workflow

Easily summon [Rubygems.org](http://rubygems.org) gem pages from the comfort of [Alfred](http://www.alfredapp.com/).

## How to Install

* clone the repo
* open `Rubygems.alfredworkflow` with Alfred

## How to Use

Within Alfred:

    gem name_of_gem
